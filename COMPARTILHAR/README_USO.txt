GUIA DE USO - PACOTE DE DASHBOARDS E RELATORIOS
================================================

1) O QUE ESTE PACOTE CONTEM
- Pacote_HTML_Offline.zip
- Dashboards e relatorios em HTML
- Biblioteca local do Plotly (libs/plotly-2.35.2.min.js)

2) REQUISITOS
- Qualquer computador com navegador moderno:
  Google Chrome, Microsoft Edge ou Mozilla Firefox.
- Nao precisa instalar Python.
- Nao precisa internet para visualizar os dashboards.

3) COMO ABRIR (PASSO A PASSO)
1. Copie o arquivo Pacote_HTML_Offline.zip para o computador de destino.
2. Extraia o .zip em uma pasta (nao abra os arquivos diretamente dentro do .zip).
3. Entre na pasta extraida.
4. Abra o arquivo Central_Biorremediacao.html no navegador.
5. Use os links internos para abrir os demais dashboards e relatorios.
6. Se preferir, o Relatorio_Final_Biorremediacao.html continua disponivel como narrativa completa.

4) ARQUIVOS PRINCIPAIS
- Central_Biorremediacao.html (ponto de entrada recomendado)
- Relatorio_Final_Biorremediacao.html
- Dashboard_Biomassa_COM_NaNO3.html
- Dashboard_PAM.html
- Dashboard_Consolidado_Biorremediacao.html
- Dashboard_Biorremediacao.html
- Relatorio_PostHoc.html

5) USO DO TEMA CLARO/ESCURO
- Clique no botao de tema no topo da pagina.
- O tema escolhido fica salvo no navegador daquele computador.

6) IMPORTANTE
- Mantenha a estrutura da pasta como veio no pacote.
- Nao mover/deletar a pasta "libs".
- Se a pasta "libs" nao estiver presente, os graficos nao carregam.

7) PROBLEMAS COMUNS E SOLUCOES
Problema: pagina abre sem grafico.
Solucao:
- Verifique se extraiu o .zip completo.
- Confirme se existe a pasta "libs" junto dos arquivos HTML.
- Reabra o arquivo no navegador.

Problema: links entre paginas nao abrem.
Solucao:
- Abra os arquivos a partir da mesma pasta extraida.
- Evite renomear arquivos HTML.

Problema: layout estranho ou incompleto.
Solucao:
- Teste em Chrome ou Edge atualizados.
- Feche e abra o navegador novamente.

8) COMPARTILHAMENTO RECOMENDADO
- Compartilhe apenas o arquivo Pacote_HTML_Offline.zip.
- O destinatario deve extrair antes de usar.

9) SUPORTE
Se precisar, abra primeiro o Relatorio_Final_Biorremediacao.html e valide:
- Tema claro/escuro funcionando
- Graficos carregando
- Links abrindo os demais paineis

Fim.

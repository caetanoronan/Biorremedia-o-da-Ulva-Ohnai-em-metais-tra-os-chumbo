Guia de Uso

Pacote de dashboards e relatorios HTML para abertura local, com e sem internet.

1. O que este pacote contem

  • Arquivos de dashboards e relatorios em HTML.
  • Biblioteca local do Plotly em libs/plotly-2.35.2.min.js.
  • Navegacao entre paginas por links internos.

2. Requisitos

  • Navegador moderno: Chrome, Edge ou Firefox.
  • Nao precisa instalar Python.
  • Nao precisa internet para visualizar os graficos.

3. Como abrir

  1. Copie o arquivo zip para o computador de destino.
  2. Extraia todo o conteudo em uma pasta.
  3. Abra Relatorio_Final_Biorremediacao.html no navegador.
  4. Use os links internos para abrir os demais arquivos.

4. Arquivos principais

  • Relatorio_Final_Biorremediacao.html (entrada recomendada)
  • Dashboard_Biomassa_COM_NaNO3.html
  • Dashboard_PAM.html
  • Dashboard_Consolidado_Biorremediacao.html
  • Dashboard_Biorremediacao.html
  • Relatorio_PostHoc.html

5. Tema claro e escuro

Clique no botao de tema no topo da pagina. A preferencia fica salva no navegador daquele computador.

6. Cuidados importantes

  • Mantenha a estrutura original da pasta.
  • Nao remover a pasta libs.
  • Sem libs, os graficos nao carregam.

7. Problemas comuns e solucoes

Pagina abre sem grafico: confirme se extraiu o zip completo e se a pasta libs existe.

Links nao abrem: mantenha todos os HTML na mesma pasta e nao renomeie arquivos.

Layout estranho: teste em Chrome ou Edge atualizados e reabra o navegador.

8. Compartilhamento recomendado

Compartilhe o arquivo zip final. O destinatario deve extrair antes de usar.

9. Checklist rapido

  • Tema claro e escuro funcionando.
  • Graficos carregando normalmente.
  • Links internos abrindo os demais paineis.
